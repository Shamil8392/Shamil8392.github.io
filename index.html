Начните с архива данных

Перед удалением запросите архив на каждой платформе. Так вы получите полный список того, что нужно удалить.

⚡

Не торопитесь

Платформы блокируют быстрые массовые действия. Делайте паузы по 2-3 секунды между удалениями, чтобы не попасть в временный бан.

👁

Закрытый аккаунт — первый шаг

Пока идёт очистка, переведите аккаунты в закрытый режим. Это скроет контент от посторонних на время работы.

🗂

Системный подход

Используйте трекер выше. Отмечайте платформы по мере завершения, чтобы не упустить ни одну.

Скрипты для браузера
// JavaScript — Консоль браузера (F12) 
// Скрипт для VK — прокрутка и удаление лайков со страницы ленты
// Откройте vk.com/feed?section=likes, затем вставьте в консоль (F12)

async function removeLikesVK() {
  const delay = ms => new Promise(r => setTimeout(r, ms));
  let count = 0;

  while (true) {
    // Находим все кнопки "лайк" (активные)
    const likeButtons = document.querySelectorAll(
      '.PostButtonReactions--liked, .like_btn.active'
    );

    if (likeButtons.length === 0) {
      console.log(`✅ Готово! Снято лайков: ${count}`);
      break;
    }

    for (const btn of likeButtons) {
      btn.click();
      count++;
      console.log(`❌ Лайк снят #${count}`);
      await delay(1500); // Пауза 1.5 сек
    }

    // Прокрутить страницу вниз для загрузки новых
    window.scrollBy(0, 1000);
    await delay(2000);
  }
}

removeLikesVK();
// JavaScript — YouTube Liked Videos 
// Скрипт для YouTube — удаление видео из "Понравившихся"
// Откройте youtube.com/playlist?list=LL → вставьте в консоль (F12)

async function removeYTLikes() {
  const delay = ms => new Promise(r => setTimeout(r, ms));
  let removed = 0;

  while (true) {
    // Меню трёх точек у первого видео в списке
    const menuBtn = document.querySelector(
      'ytd-playlist-video-renderer:first-child #button[aria-label]'
    );

    if (!menuBtn) {
      console.log(`✅ Завершено. Удалено: ${removed}`);
      break;
    }

    menuBtn.click();
    await delay(800);

    // Найти пункт "Удалить из Понравившихся"
    const items = [...document.querySelectorAll('ytd-menu-service-item-renderer')];
    const removeItem = items.find(i => i.innerText.includes('Удалить'));

    if (removeItem) {
      removeItem.click();
      removed++;
      console.log(`🗑 Удалено #${removed}`);
    }

    await delay(1200);
  }
}

removeYTLikes();

Как использовать скрипты

  1. Откройте нужный сайт в браузере (Chrome или Firefox)
  2. Нажмите F12 для открытия инструментов разработчика
  3. Перейдите на вкладку Console (Консоль)
  4. Скопируйте нужный скрипт кнопкой «Копировать»
  5. Вставьте скрипт в поле консоли и нажмите Enter
  6. Следите за прогрессом в консоли — каждое действие логируется
  7. Чтобы остановить — обновите страницу (F5)
⚠️

Скрипты работают только на страницах, где вы авторизованы. Не запускайте их на чужих сайтах или незнакомых ссылках.

СоцЧистка — локальный инструмент. Никакие данные не передаются. Всё работает в вашем браузере.

🔵 VK — Массовое удаление

ВКонтакте позволяет управлять активностью через консоль браузера. Скрипт ниже автоматически снимает лайки на загруженной странице.

  1. Откройте vk.com/feed?section=likes
  2. Нажмите F12 → перейдите в «Консоль»
  3. Скопируйте скрипт из раздела «Скрипты для браузера»
  4. Вставьте и нажмите Enter. Скрипт запустится автоматически
  5. Для постов: зайдите на свою страницу, откройте консоль и аналогично

🟠 Одноклассники — Инструкция

В OK нет отдельного раздела с лайками, но есть несколько способов найти активность:

  1. Зайдите в «Мой профиль» → «Активность» — там история действий
  2. Для удаления постов: профиль → найдите пост → три точки → «Удалить»
  3. Фото: Фото → выберите → «Удалить фото»
  4. Комментарии: найдите через уведомления или историю
  5. Запросить данные: ok.ru/settings → «Скачать данные»

⚫ TikTok — Удаление лайков

TikTok ограничивает массовые действия, но вот самый быстрый способ:

  1. Откройте приложение TikTok → Профиль
  2. Нажмите на вкладку ♥ (Понравившееся)
  3. Для каждого видео: нажмите и удерживайте → «Убрать из понравившихся»
  4. Для удаления своих видео: три точки на видео → «Удалить»
  5. Через веб: tiktok.com/profile → раздел Liked Videos
  6. Можно запросить архив данных в Настройки → Конфиденциальность → Данные

🟣 Instagram — Управление активностью

Instagram скрывает лайки — лучший способ найти их через архив данных:

  1. Настройки → «Ваша деятельность» → «Скачать данные»
  2. Выберите JSON формат, укажите email → «Запросить скачивание»
  3. Через 24-48 ч придёт файл — в нём все лайки и комментарии
  4. Для удаления постов: профиль → три точки → «Удалить»
  5. Сторис: автоматически удаляются через 24 ч, но архив хранится
  6. Комментарии: Настройки → Ваша деятельность → Комментарии

🔷 Telegram — Удаление контента

В Telegram можно удалить сообщения у всех участников без ограничений по времени (в личных чатах):

  1. В личном чате: зажмите сообщение → «Удалить» → «Для всех»
  2. В группах: можно удалить только свои сообщения (не у всех)
  3. Очистить всю историю с собой: чат «Избранное» → очистить
  4. Удалить аккаунт полностью: my.telegram.org → Delete Account
  5. Экспорт данных: Настройки → Конфиденциальность → Экспорт данных
